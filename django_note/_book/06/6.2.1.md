# 数据表设计

- 分类表 (cate)
  - id  分类id
  - name  分类名称
  - status 分类状态
  - addtime 添加时间
- 图书表 (book)
  - id  图书id
  - name 图书名称
  - price  图书价格
  - img    图书封页
  - status  图书状态
  - addtime   添加时间
- 用户表 (user)
  - id    用户id
  - name   用户名
  - password   密码
- 购物车表(cart)
  - id 自增id
  - book_id  图书id
  - user_id 用户id
